## 执行目标
- 启动本地 Next.js 开发服务器并获得预览 URL。
- 在首页抓取首个文章链接，访问对应详情页，验证 200 状态与页面渲染。

## 步骤
1. 在项目根目录执行 `npm run dev` 启动服务器（端口默认 `http://localhost:3000`，以实际输出为准）。
2. 等待服务就绪后，打开首页 `/`，抓取页面中第一个 `/article/yyyy/mm/dd/title` 链接。
3. 访问该详情链接：
   - 验证响应状态为 200；
   - 验证页面包含文章标题与正文容器（`<h1>` 与内容区域）以确认渲染成功。
4. 将可用预览 URL 提供给你进行自测，同时给出我们自动验证的结果。

## 可能的前置检查
- 若首页无文章（数据库为空），则改为直接枚举数据库获取一篇文章的 `createdAt` 与 `title` 拼装测试 URL；若仍无数据，标记不可测试并提示需要至少一篇文章数据。

## 产出
- 启动成功的预览 URL。
- 跳转请求的实际测试链接与状态验证结果。
