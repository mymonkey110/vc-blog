# 静态生成博客首页（列表页）

## 功能描述

从数据库中读取博客数据，生成博客首页（列表页）。

## 实现与技术细节

- 数据库操作使用prisma ORM
- 连接数据库：使用环境变量(env文件中的DATABASE_URL)中的数据库连接字符串，连接到neon数据库
- 查询文章表(article)，提取所有文章的以下字段
    - title：文章标题
    - createdAt：文章发布日期
    - description：文章描述 
- 生成首页列表：
    - 参考UI:`ui/前台/博客首页`
    - 首页文件生成到`src/pages/index.tsc`中
    - 排序按发布时间倒叙排列
    - 分页细节：
        - 每页显示10篇文章
        - 分页组件生成到`src/components/pagination.tsx`中
    - 其他
        - 图片部分留空
        - 搜索部分先不实现
