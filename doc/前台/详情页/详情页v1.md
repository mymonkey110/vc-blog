# 静态生成博客文章详情页

## 功能描述

从数据库中读取博客数据，生成博客文章详情页。

## 实现与技术细节

- 数据库操作使用prisma ORM
- 详情页使用Next.js的SSG(静态站点生成)功能
- 连接数据库：使用环境变量(env文件中的DATABASE_URL)中的数据库连接字符串，连接到neon数据库
- 查询文章表(article)，提取所有文章的以下字段
    - title：文章标题
    - createdAt：文章发布日期
    - description：文章描述 
    - content：文章内容
- 生成文章详情页：
    - 参考UI:`ui/前台/文章详情页`
    - 路由设计：
        - 文章详情页的路由为`/article/[yyyy]/[mm]/[dd]/[title]`，其中`[yyyy]`为文章发布年份，`[mm]`为文章发布月份，`[dd]`为文章发布日期，`[title]`为文章标题。
    - 文章内容(content)是Markdown格式，需要使用Markdown解析器解析为HTML格式。
    - 文章内容中图片的解析
      - 本地图片：使用Next.js的图片组件(Image)解析本地图片，图片路径为`/public/images/[filename]`，其中`[filename]`为图片文件名。
      - 网络图片：直接使用图片URL解析，无需特殊处理。
    - 文章中引用的代码片段需要高亮展示
      - 使用Next.js的代码高亮组件(CodeBlock)解析代码片段，代码高亮使用Prism.js库。
      - 代码展示样式与文章之间需要区分度
      - 代码片段增加右上角增加一件复制功能
    - 样式
      - 字体：
        - 文章内容字体：使用系统默认字体
        - 代码字体：使用等宽字体(如Monaco, Consolas等)
        - 尽量采用系统自带字体，避免额外下载
    - 其他
        - 移除`作者`，`文章标签`
        - 评论部分留空，以后再实现
