# 项目介绍

这是一个博客项目的前端部分，包含首页、文章详情页、分类页、标签页、关于页等功能。

## 网页地图

- 首页：展示最新的文章列表
- 文章详情页：展示文章的详细内容
- 分类页：展示不同分类下的文章列表
- 标签页：展示不同标签下的文章列表
- 关于页：展示个人介绍和联系方式

## UI 设计

- 响应式设计：适配不同屏幕尺寸，包括手机、平板和桌面端
- 组件库：尽量不要使用第三方组件库，使用 React 组件库中的组件，避免引入额外的依赖；或者自己实现
- 图标库：使用 Font Awesome 图标库，提供常用的图标
- 颜色方案：使用 Tailwind CSS 提供的颜色方案，保持一致的颜色风格
- 交互稿：参考ui目录下的交互稿，实现对应的交互效果

# 技术栈

- 语言：TypeScript
- 框架：Next.js
- 样式：Tailwind CSS

# 代码规范

- 代码格式化：Prettier
- 代码检查：ESLint
- 提交规范：Conventional Commits

# 代码原则

- 组件化：每个功能都封装为一个组件，组件之间通过 props 进行通信
- 状态管理：使用 React Hooks 进行状态管理
- 路由管理：使用 Next.js 内置的路由功能
- 接口调用：使用 Axios 进行接口调用
- 错误处理：统一处理接口调用错误，展示错误信息
- 减少重复代码：使用 React 组件库中的组件，避免重复编写相同的代码
- 代码注释：对每个组件和函数进行注释，说明其功能和参数
